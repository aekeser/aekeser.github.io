# Ahmet Emre Keser – Academic Website

This repository contains the source code for my personal academic website:

https://aekeser.github.io/

The website was designed and implemented by Ahmet Emre Keser using Jekyll, HTML, and CSS, and is hosted through GitHub Pages.

The site presents information about my research, teaching, publications, academic projects, and professional activities in linguistics, language processing, aphasia, and computational linguistics.

## Languages

The website is available in both English and Turkish.
